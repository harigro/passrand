# PassRand

PassRand adalah aplikasi untuk mempermudah membuat kata sandi secara acak

## Instalasi Depedensi

```bash
pip install -r requirements.txt
```

## Menjalankan Aplikasi

* Linux dan MacOS
  
  * ```bash
    python3 main.py
    ```

* Windows OS
  
  * ```bash
    python main.py
    ```
